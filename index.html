My Projects

Gotta Lotto

GottaLotto is a guessing game app for lotto enthusiasts, dreamers, and people that want to have plain old fun in guessing the winning numbers for each Saturday's PowerBall drawing. It allows users to submit their guesses for the upcoming drawing and a text area where users can include their lottery experience message. Submitted guesses are scored based on how close they get from the Powerball winning numbers

Start Small

Start small is an application for users who love to help in funding classroom projects. The app shows classroom project listings that they can review based on 1 to 5 stars. It also provides an external link to the specific projects donating site.

Scrabble Solver

Scrabble solver is an application that produces valid words based on provided scrabble letters or tiles. It shows dictionary definitions/etymologies when clicking on valid word results. It allows users to sort word results by score or length. Worked and collaborated in a team setting.

COVID-19 Quizapp
